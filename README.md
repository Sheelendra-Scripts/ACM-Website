üíª Getting Started
Prerequisites

Ensure you have Node.js installed (v18 or higher recommended).
Installation

Clone the repository:

git clone https://github.com/Arsh199965/ACM-Website.git
cd ACM-Website/client

Install dependencies:

npm install
# or
yarn install
# or
pnpm install

Run the development server:

npm run dev

    Open http://localhost:3000 with your browser to see the result.

Building for Production

To create an optimized static build:

npm run build

This will generate an out folder containing the static assets, ready for deployment.
