# GGSIPU EDC ACM Student Chapter Website

Official website of the **GGSIPU East Delhi Campus ACM Student Chapter**.  
This platform serves as a central hub for events, technical blogs, projects, and team updates, empowering students to innovate and collaborate in the field of computing.

Live Website: https://usar.acm.org  

---

## Features

**Dynamic Hero Sections**  
- Smooth parallax effects  
- Gradient-based animations  

**Events Dashboard**  
- Browse upcoming and past events  
- Detailed event insights and highlights  

**Tech Blogs**  
- Articles written by community members  
- Covering emerging and core technologies  

**Team Showcase**  
- Faculty mentors  
- Core team members and visionaries  

**Projects Gallery**  
- Innovative projects built by chapter members  

**High Performance**  
- Lighthouse score: 100  
- SEO-optimized and fast  

**Dark Mode Aesthetic**  
- Futuristic, minimal, and professional UI  

---

## Tech Stack

| Category | Technology |
|--------|------------|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Animations | Framer Motion |
| 3D Elements | Three.js / React Three Fiber |
| Deployment | Static Export (Hosted on cPanel) |
